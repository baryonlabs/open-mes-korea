# MVP 범위

## 대상 사용자

- 생산관리자
- 품질관리자
- 자재/창고 담당자
- 현장 작업자
- 시스템 관리자

## 필수 기능

### 기준정보

- 품목 관리
- BOM 관리
- 공정 관리
- 라우팅 관리
- 설비 관리
- 작업자 관리

### 생산관리

- 작업지시 생성
- 작업지시 상태 관리
- 공정별 작업 시작
- 공정별 작업 종료
- 양품 수량 입력
- 불량 수량 입력
- 불량 유형 기록

### LOT 추적

- 원자재 LOT 등록
- 공정 투입 LOT 기록
- 반제품/제품 LOT 생성
- LOT genealogy 조회

### 현장 화면

- 오늘 작업 목록
- 작업 시작/중지/완료
- 생산수량 입력
- 불량수량 및 사유 입력
- 바코드 또는 QR 입력 준비

### 조회 화면

- 작업지시 진행 현황
- 공정별 생산 실적
- 불량 현황
- LOT 이력
- 품목별 재고 흐름

### 관리자

- 사용자 관리
- 권한 관리
- 코드/마스터 데이터 관리
- 감사 로그 조회

## 제외 범위

초기 MVP에서는 아래를 제외합니다.

- 회계
- 급여
- 복잡한 구매/영업
- 고급 생산계획 최적화
- 설비 자동 수집 전체
- 모바일 앱 네이티브 구현

## 우선 구현 흐름

1. 품목, 공정, BOM, 라우팅
2. 작업지시
3. 공정 실적 입력
4. LOT 투입 및 생성
5. 생산현황 조회
6. AI 조회/요약 API
