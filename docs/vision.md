# 프로젝트 비전

Open MES Korea는 한국 제조 현장의 언어와 업무 흐름을 기본값으로 삼는 오픈소스 MES입니다.

이 프로젝트의 집중 영역은 ERP 전체가 아니라 현장 실행, LOT 추적, AI context/approval layer입니다.

## 핵심 원칙

1. 현장 우선

   작업자 화면은 빠르고 단순해야 합니다. 많은 설명보다 큰 버튼, 명확한 상태, 실수 방지가 중요합니다.

2. 이력 우선

   생산 시스템은 현재 상태만 저장하면 안 됩니다. 누가, 언제, 어떤 작업지시에서, 어떤 LOT를 투입했고, 어떤 결과가 나왔는지 추적 가능해야 합니다.

3. 한국어 우선

   메뉴, 업무 용어, 도움말, 설치 문서는 한국어를 기본으로 합니다. 영어 번역은 확장 항목입니다.

4. 작은 공장도 설치 가능

   초기 버전은 Docker Compose로 단일 서버에 설치할 수 있어야 합니다.

5. AI가 읽기 좋은 MES

   AI 기능은 나중에 붙이는 장식이 아니라 데이터 모델, 감사 로그, 권한, API 설계에 처음부터 반영합니다.

6. ERP와 경쟁하지 않는 MES

   회계, 인사, 영업, 구매 전체 기능을 직접 만들기보다 ERP와 연동 가능한 현장 실행 시스템을 먼저 완성합니다.

## 만들지 않을 것

초기 단계에서는 아래를 목표로 하지 않습니다.

- 모든 업종을 만족하는 초대형 MES
- 복잡한 APS 최적화
- 자체 ERP 전체 기능
- 설비 PLC/SCADA 범용 드라이버 전체 구현
- AI가 승인 없이 생산 데이터를 변경하는 기능

## 성공 기준

초기 성공 기준은 단순합니다.

- 작은 제조사가 품목, 공정, 작업지시, 실적, LOT 이력을 관리할 수 있다.
- 작업자는 태블릿 또는 현장 PC에서 공정 실적을 입력할 수 있다.
- 관리자는 생산 진행과 불량 현황을 한국어로 조회할 수 있다.
- AI는 데이터를 설명하고 제안할 수 있지만, 중요한 변경은 사람의 승인을 받는다.
