# 제조사 도입 검토 가이드

이 폴더는 제조 회사가 Open MES Korea 도입 여부를 판단하기 전에 확인해야 할 내용을 정리한다.

대상 독자는 개발자가 아니라 다음 역할이다.

- 대표/공장장
- 생산관리 책임자
- 품질관리 책임자
- 자재/창고 책임자
- 설비/보전 담당자
- 사내 IT 담당자
- 스마트공장 공급기업/SI

## 기본 판단

Open MES Korea는 다음 회사에 특히 적합하다.

- 엑셀, 수기, 메신저로 생산 실적을 관리한다.
- 작업지시 진행 상태가 실시간으로 보이지 않는다.
- LOT 추적이 필요하지만 현재 이력이 흩어져 있다.
- 불량 발생 시 원인, 공정, 작업자, 자재 LOT 추적이 어렵다.
- ERP는 있지만 현장 실행 데이터가 약하다.
- 상용 MES 도입 비용이나 벤더 종속이 부담스럽다.
- AI를 쓰고 싶지만 먼저 제조 데이터를 정리해야 한다.

반대로 다음 상황이면 바로 도입보다 준비가 먼저다.

- 품목 코드와 공정 기준이 전혀 정리되어 있지 않다.
- 현장 작업자가 시스템 입력을 할 시간이나 장비가 없다.
- 경영진이 데이터 기반 운영보다 단순 보고서만 원한다.
- 기존 ERP/MES/설비 시스템과의 책임 경계가 불명확하다.
- LOT 추적이 필요 없고 단순 일일 생산량만 보면 된다.

## 도입 원칙

처음부터 모든 공장 업무를 바꾸려 하지 않는다.

권장 시작 범위:

```text
작업지시
→ 현장 작업 시작/종료
→ 생산수량/불량수량 입력
→ 자재 LOT 투입
→ 제품 LOT 생성
→ 생산현황/LOT 이력 조회
```

이 흐름이 안정되면 설비 데이터, OEE, AI 분석, ERP 연동을 단계적으로 붙인다.
