# 도입 적합성 체크리스트

이 문서는 Open MES Korea가 현재 공장에 적합한지 빠르게 판단하기 위한 체크리스트다.

## 1. 생산 운영 방식

| 질문 | 예 | 아니오 | 판단 |
|---|---:|---:|---|
| 작업지시를 생산관리자가 발행한다 |  |  | 예면 적합 |
| 공정별 작업 시작/종료를 기록할 필요가 있다 |  |  | 예면 적합 |
| 작업지시 진행 상황을 실시간 또는 당일 단위로 보고 싶다 |  |  | 예면 적합 |
| 다품종 소량생산 또는 job shop 성격이 있다 |  |  | 예면 적합 |
| 제품별 공정 순서가 어느 정도 정해져 있다 |  |  | 예면 적합 |
| 생산량만 일 단위로 합산하면 충분하다 |  |  | 예면 MES 필요성이 낮음 |

## 2. LOT/추적성 필요성

| 질문 | 예 | 아니오 | 판단 |
|---|---:|---:|---|
| 원자재 LOT를 제품 LOT와 연결해야 한다 |  |  | 예면 매우 적합 |
| 고객 클레임 시 어떤 자재가 투입됐는지 찾아야 한다 |  |  | 예면 매우 적합 |
| 식품, 화장품, 전자부품, 의료기기, 자동차부품 등 추적성이 중요하다 |  |  | 예면 매우 적합 |
| 불량 발생 시 같은 LOT 제품을 찾아야 한다 |  |  | 예면 매우 적합 |
| LOT 구분 없이 총 재고만 관리한다 |  |  | 아니오면 준비 필요 |

## 3. 현장 입력 가능성

| 질문 | 예 | 아니오 | 판단 |
|---|---:|---:|---|
| 현장에 PC, 태블릿, 키오스크 중 하나를 둘 수 있다 |  |  | 예면 적합 |
| 작업자가 바코드/QR 스캔 또는 버튼 입력을 할 수 있다 |  |  | 예면 적합 |
| 작업자가 종이 작업일보를 작성하고 있다 |  |  | 예면 대체 가능 |
| 현장 네트워크가 불안정하다 |  |  | 오프라인/edge 고려 필요 |
| 작업 중 시스템 입력이 생산성을 크게 방해한다 |  |  | UX 재설계 필요 |

## 4. 기존 시스템

| 질문 | 예 | 아니오 | 판단 |
|---|---:|---:|---|
| ERP가 이미 있다 |  |  | MES 연동형 도입 적합 |
| ERP는 있지만 현장 실적 입력은 약하다 |  |  | 매우 적합 |
| 기존 MES가 있지만 유지보수나 커스터마이징이 어렵다 |  |  | 전환/보완 검토 |
| 설비 데이터 수집 시스템이 이미 있다 |  |  | 연동 방식 검토 |
| 모든 업무를 하나의 상용 ERP에서 이미 잘 처리한다 |  |  | 도입 필요성 낮음 |

## 5. AI native 준비도

| 질문 | 예 | 아니오 | 판단 |
|---|---:|---:|---|
| 생산/불량/LOT 데이터를 자연어로 조회하고 싶다 |  |  | AI context API 필요 |
| AI가 작업지시 변경이나 품질 이슈 초안을 제안하면 도움이 된다 |  |  | approval workflow 적합 |
| AI가 승인 없이 데이터를 바꾸면 안 된다 |  |  | Open MES Korea 방향과 일치 |
| 현재 데이터가 흩어져 있어 AI 분석이 어렵다 |  |  | MES core부터 필요 |

## 판정

| 예 답변 수 | 판정 |
|---:|---|
| 15개 이상 | 도입 검토 가치가 높음 |
| 10~14개 | 부분 도입 또는 PoC 권장 |
| 5~9개 | 기준정보 정리 후 재검토 |
| 4개 이하 | 현재는 MES보다 업무 표준화가 먼저 |
