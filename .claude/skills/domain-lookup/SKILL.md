사용자가 `/domain-lookup` 또는 특정 엔티티명을 언급하면 `docs/domain-model.md`를 읽고 해당 엔티티의 필드와 상태 머신을 정리해서 보여줍니다.

## 수행 방법

1. `docs/domain-model.md`를 읽는다.
2. 사용자가 요청한 엔티티(예: WorkOrder, MaterialLot)를 찾는다.
3. 필드 목록과 해당 엔티티의 상태 머신(있는 경우)을 표 형식으로 출력한다.
4. 관련 엔티티(외래키로 연결된 것)도 함께 안내한다.

## 출력 형식

- 엔티티명 (한국어 설명)
- 필드 목록 (필드명 | 설명)
- 상태 머신: 허용된 전이 목록
- 관련 엔티티

## 주의사항

- 상태 머신에 없는 전이를 임의로 추가하거나 제안하지 않는다.
- `AuditLog`와 `AiInteraction`은 모든 엔티티와 연관될 수 있음을 항상 언급한다.
