# Security Policy

Open MES Korea is currently in early implementation and has no production
release.

Do not report security vulnerabilities in a public issue. Contact the
maintainers through the private contact form:

https://openmeskorea.org/#contact

Include:

- affected commit or component
- reproduction steps
- expected impact
- suggested mitigation, if available

Do not include real factory credentials, customer data, PLC addresses, device
keys, or production records.
